Prompts

Prompts para negocios: ideas utiles para vender, organizar y atender clientes

Un buen prompt no hace magia, pero ayuda a ordenar informacion, ahorrar tiempo y crear mejores primeros borradores.

1. Prompt para entender a tu cliente

Actua como consultor de marketing. Analiza este producto, define tres tipos de cliente probable, sus problemas principales y objeciones antes de comprar: [describe tu producto].

2. Prompt para responder mensajes

Convierte este mensaje de cliente en una respuesta clara, amable y profesional. Manten el tono cercano, responde la duda principal y agrega un siguiente paso: [pega el mensaje].

3. Prompt para ideas de contenido

Genera 20 ideas de contenido para un negocio de [sector]. Clasificalas por etapa: descubrimiento, confianza, comparacion y compra.

4. Prompt para mejorar procesos

Revisa este proceso interno y detecta pasos repetidos, posibles automatizaciones y riesgos de comunicacion: [describe el proceso].

Conclusion

Los prompts funcionan mejor cuando das contexto, objetivo, restricciones y formato esperado. Despues, revisa todo con criterio humano antes de publicar o enviar.
